## Project Overview
Private beta bootstrap of gaming-focused Web3 wallet with core security system design

## Key Responsibilities & Contributions
- Designed E2E encryption architecture with password-private key one-to-one mapping
- Implemented secure key recovery system based on non-custodial model
- Developed multi-chain wallet infrastructure
- Conducted security audits and cryptographic protocol verification

## Tech Stack
- **Frontend**: Chrome Extension
- **Blockchain**: Ethereum, Polygon, Multi-chain Support
- **Cryptography**: E2E Encryption, Key Management
- **Backend**: Node.js, TypeScript
- **Security**: HSM, Secure Enclave, Key Recovery
