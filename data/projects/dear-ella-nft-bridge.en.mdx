## Project Overview
Bridge server system that converts in-game assets from mobile game Dear, Ella into XPLA blockchain NFTs and maintains synchronization

## Key Responsibilities & Contributions
- Designed and implemented real-time on-chain/off-chain synchronization server
- Developed NFT minting and transfer logic for game assets
- Built blockchain transaction monitoring and state synchronization system
- Led DevOps processes and infrastructure management

## Tech Stack
- **Blockchain**: XPLA, Smart Contracts
- **Backend**: Node.js, TypeScript, WebSocket
- **Database**: MSSQL, Redis
- **DevOps**: Docker, Kubernetes, CI/CD
