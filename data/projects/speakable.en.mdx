## Key Responsibilities & Contributions
- Prototyped real-time voice conversation AI system
- Integrated Hume AI EVI and Gemini 2.5 Flash
- Implemented personalized conversation context maintenance system
- Developed adaptive response generation logic based on user level

## Tech Stack
- **AI**: Hume AI SDK (EVI), Gemini 2.5 Flash
- **Frontend**: Next.js, React, TypeScript
- **UI**: Tailwind CSS
- **Infrastructure**: Vercel (Serverless)
