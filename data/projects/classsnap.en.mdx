## Project Overview
AI learning platform that systematically improves English reading comprehension based on pedagogical methods

## Key Responsibilities & Contributions
- Designed reading skill learning system based on pedagogical theory
- Implemented Google Gemini-based adaptive difficulty text generation and recommendation engine
- Developed learning progress tracking and personalized feedback system
- Designed and implemented teacher-student dashboard

## Tech Stack
- **AI**: Google Gemini
- **TTS**: Azure Speech
- **Frontend**: Next.js, React, TypeScript
- **Backend**: Supabase
- **UI**: Tailwind CSS, Radix UI, Lucide React
- **Infrastructure**: Vercel
