# 환영합니다!

안녕하세요! 다국어 블로그에 오신 것을 환영합니다. 이 블로그는 **영어**와 **한국어**를 모두 지원합니다.

## 기능

- 🌐 **다국어 지원**: 헤더에서 영어와 한국어를 전환할 수 있습니다
- 🎨 **모던 디자인**: 다크 모드를 지원하는 아름답고 반응형 디자인
- 📝 **MDX 지원**: Markdown과 JSX 컴포넌트로 포스트 작성
- 🔍 **검색**: 원하는 내용을 빠르게 찾을 수 있습니다
- 🏷️ **태그**: 주제별로 포스트 정리

## 언어 전환 방법

헤더의 언어 전환 버튼을 사용하여 언어를 전환할 수 있습니다. 언어 설정은 브라우저에 저장됩니다.

## 기술 스택

이 블로그는 다음 기술로 구축되었습니다:

- **Next.js 16**: 최신 React 프레임워크
- **Tailwind CSS**: 유틸리티 우선 CSS 프레임워크
- **Contentlayer**: MDX를 위한 콘텐츠 관리
- **TypeScript**: 타입 안전 개발

방문해 주셔서 감사합니다!
